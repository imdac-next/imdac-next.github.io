# Sass Parent Selector

Watch "Referencing parent selectors with &" on LinkedIn Learningopen in new window

The parent selectoropen in new window (&) is a special selector found in Sass and is used to refer to the outer selector when nesting. The parent selector makes it possible to neatly add and complex selectors in an easy to read fashion.

The parent selector can be used in conjunction with pseudo-classes. For example, it could be used to style navbar links.

$yellow: #FFBA00;
$border-color: #D14348;

.link {
  color: black;
  border-right: 1px solid $border-color;

  &:hover, &.active {
    background-color: $yellow;
    font-weight: bold;
  }

  &:last-of-type {
    border-right: 0;
  }
}
