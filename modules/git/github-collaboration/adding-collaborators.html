# Adding Collaborators

GitHub repositories can be used with almost any kind of projects that requires the tracking of text files, and creating a repository is simple process.

  1. After creating an account and signing into GitHub, a new repository can be created by clicking the + drop-down menu and selecting New repository.

  1. Provide a name for the repository that is short and descriptive. The repository name must unique within the selected account.
